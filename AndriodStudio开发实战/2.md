### 2.1.1 像素
```java
常用px db sp 三种像素单位

dp:与物理设备无关与屏幕尺寸相关,note: 一样屏幕大小dp 大小一样尺寸一样
sp:专门用来设置字体大小 sp 与手机设置里可以调整字体大小(小 普通 大 超大) 相关


dp 和 px 之间的关系
1.获取像素密度
  density=1.0 px : dp =1:1
  density=1.5 px : dp =2:3

2.dp  px 相互转换
dp->px  dp*density+0.5f
px->dp  px/density+0.5f
```
